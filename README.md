# tjenamors.se

Synthwave-themed radio station website. Built with vanilla HTML, CSS & JS —
no frameworks, no build step (just a file copy).

## Develop

```bash
node scripts/build.mjs
```

Output lands in `dist/`. Cloudflare Pages auto-deploys from `main`.

## E2E

```bash
npm install
node tests/<name>.mjs
```
